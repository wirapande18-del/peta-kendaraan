PETA KENDARAAN V10 - AI OFFLINE GRATIS

Fitur baru:
1. Pembuat pesan WhatsApp offline tanpa OpenAI API.
2. Tidak memerlukan API key, saldo, kartu, atau pembayaran.
3. Pilihan gaya bahasa: profesional, ramah, singkat, persuasif, formal.
4. Pilihan panjang pesan: pendek, sedang, panjang.
5. Deteksi topik otomatis: oli, AC, rem, aki, ban, Body & Paint, T-Care, booking.
6. Klik Buat Pesan Offline berulang kali untuk variasi pesan berbeda.
7. Rapikan Pesan Ini bekerja sepenuhnya di browser.
8. Semua data kendaraan, peta, filter, status follow-up, dan template lama tetap dipertahankan.

Cara pakai:
- Buka Kelola Template.
- Pilih kategori.
- Isi topik/perintah.
- Pilih gaya dan panjang.
- Klik Buat Pesan Offline.
- Edit bila perlu lalu klik Simpan Template.

Catatan:
File api/ai-message.js tetap disertakan agar struktur V9 tidak rusak, tetapi fitur AI Offline V10 tidak memakainya.
