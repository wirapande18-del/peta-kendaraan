PETA KENDARAAN SERVICE V13.7

FITUR BARU: INPUT MANUAL STATUS WHATSAPP

Pilihan cepat yang tersedia:
1. Centang Satu
2. Centang Dua - Belum Dibaca
3. Dibaca - Tidak Dibalas
4. Sudah Dibalas

Cara menggunakan:
1. Klik marker kendaraan atau buka menu Follow Up.
2. Klik Input Status WA / Ubah Status.
3. Pilih salah satu tombol status WhatsApp.
4. Tanggal dan reason otomatis disiapkan.
5. Tambahkan catatan jika diperlukan.
6. Klik Simpan Follow Up.

Perbedaan status otomatis dan manual:
- WA Dibuka (Otomatis): tercatat ketika tombol WhatsApp dibuka dari aplikasi.
  Status ini tidak memastikan pesan sudah dikirim.
- Centang Satu, Centang Dua, Dibaca, dan Dibalas: dipilih manual oleh pengguna
  berdasarkan tanda yang terlihat pada WhatsApp.

Perlindungan data:
- Membuka WhatsApp lagi tidak menimpa status manual yang sudah dipilih.
- Status lama tetap dapat dibaca.
- Riwayat perubahan status tetap disimpan.
- Dashboard dan download Follow Up menggunakan status manual terbaru.

Pastikan aplikasi menampilkan Versi 13.7 setelah deployment.
