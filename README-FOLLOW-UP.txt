VERSI INI SUDAH DITAMBAHKAN REASON FOLLOW UP.

Cara upload:
1. Ekstrak ZIP.
2. Upload seluruh isi ke repository GitHub yang lama dan pilih Replace files.
3. Commit changes.
4. Tunggu Vercel deploy, lalu Ctrl+F5.

Data follow up tersimpan di browser dan terhubung berdasarkan POLICE_NO/plat nomor. Saat Excel dengan plat yang sama di-upload lagi, status follow up tetap terbaca pada perangkat/browser yang sama. Tombol Download Follow Up menghasilkan Excel kendaraan lengkap beserta kolom follow up.
