PETA KENDARAAN SERVICE V13.3
AUDIT TITIK SALAH & ANTI-CACHE

PERBAIKAN
1. Nomor versi "Versi 13.3" selalu terlihat di kanan atas aplikasi.
2. Nomor versi juga ditampilkan di popup setiap kendaraan.
3. File index.html, app.js, dashboard.js, storage.js, dan data.js memakai aturan no-cache.
4. Nama cache file diperbarui agar browser tidak menjalankan JavaScript versi lama.
5. Tombol "Periksa Titik Salah" ditambahkan pada halaman Data & Upload.
6. Tombol memeriksa seluruh koordinat terhadap kabupaten/kota dari alamat.
7. Titik DK-1482-ADU di Singaraja akan dihapus karena alamatnya Denpasar.
8. Setelah dihapus, data berstatus "perlu diproses ulang".

CARA MEMAKAI SETELAH DEPLOY
1. Pastikan kanan atas menampilkan "Versi 13.3".
2. Jika belum terlihat, deployment belum benar atau halaman masih versi lama.
3. Buka Data & Upload.
4. Klik "Periksa Titik Salah".
5. Setelah muncul jumlah titik yang dihapus, klik "Proses alamat".
6. Gunakan 250 alamat per sesi.

CATATAN
- Jangan klik Hapus pada kendaraan DK-1482-ADU.
- Tombol Periksa Titik Salah hanya menghapus koordinat yang tidak sesuai wilayah,
  bukan menghapus data customer atau riwayat follow up.
- Semua perbaikan versi 13.2 dan optimasi 7.000+ data tetap dipertahankan.
