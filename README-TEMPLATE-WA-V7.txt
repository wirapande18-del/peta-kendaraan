UPDATE V7 - TEMPLATE WHATSAPP EDITABLE

Fitur lama tetap dipertahankan.

Fitur baru:
1. Pilih template aktif dari panel Follow Up.
2. Kelola template: tambah, edit, duplikat, hapus, aktif/nonaktif.
3. Template bawaan: Reminder Service, Promo Service, Promo Ganti Oli, Follow Up Booking.
4. Variabel otomatis: {nama}, {plat}, {model}, {tahun}, {km}, {sa}, {alamat}, {tanggal}.
5. Pesan dapat diedit lagi sebelum WhatsApp dibuka tanpa mengubah template utama.
6. Import template dari Excel/CSV.
7. Export seluruh template ke Excel.
8. Template disimpan lokal di browser/perangkat, terpisah dari data kendaraan.

Format import Excel:
NAMA_TEMPLATE | KATEGORI | ISI_PESAN | AKTIF
Contoh kategori: REMINDER, PROMO, UCAPAN, LAINNYA
AKTIF: YA atau TIDAK

Cara pakai:
- Upload semua isi ZIP ke project GitHub/Vercel seperti versi sebelumnya.
- Buka menu Template Pesan WA pada panel kanan.
- Klik Kelola Template untuk mengubah isi pesan.
