PETA KENDARAAN SERVICE V13.5

PERBAIKAN CARI & PINDAHKAN TITIK

Kolom pencarian sekarang menerima:
1. Alamat lengkap.
2. Link lengkap Google Maps yang memiliki tanda @latitude,longitude.
3. Latitude dan longitude, contoh: -8.4302136, 115.3316231
4. Urutan longitude,latitude juga akan dibalik otomatis jika terdeteksi.

Cara memakai koordinat:
1. Klik marker kendaraan.
2. Klik Cari & Pindahkan Titik.
3. Tempel koordinat: -8.4302136, 115.3316231
4. Klik Cari / Gunakan.
5. Peta kecil langsung berpindah ke koordinat tersebut.
6. Periksa marker, kemudian klik Simpan Titik Ini.

Cara memakai link Google Maps:
1. Salin link lengkap dari Google Maps.
2. Tempel di kolom pencarian.
3. Klik Cari / Gunakan.
4. Jika link berisi koordinat, marker langsung dipindahkan.

Catatan:
- Link pendek seperti maps.app.goo.gl mungkin belum membawa angka koordinat.
  Jika tidak terbaca, buka link tersebut lalu salin link lengkap atau angka
  latitude dan longitude.
- Koordinat di luar wilayah Bali akan ditolak untuk mencegah salah titik.
- Pastikan aplikasi menampilkan Versi 13.5 setelah deployment.
