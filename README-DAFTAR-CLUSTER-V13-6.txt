PETA KENDARAAN SERVICE V13.6

FITUR BARU: DAFTAR KENDARAAN PADA ANGKA KELOMPOK

Cara menggunakan:
1. Buka menu Peta Kendaraan.
2. Klik angka kelompok pada peta, misalnya angka 31.
3. Akan muncul daftar seluruh kendaraan dalam kelompok tersebut.
4. Gunakan kolom pencarian untuk mencari plat, customer, SA, model, atau alamat.
5. Klik salah satu kendaraan untuk menuju marker dan membuka detail kendaraan.
6. Jika ingin memperbesar seluruh area kelompok, klik tombol Perbesar Area.

Untuk kelompok besar:
- Daftar dibagi menjadi 100 kendaraan per halaman agar tetap cepat.
- Tombol Sebelumnya dan Berikutnya digunakan untuk berpindah halaman.

Fitur sebelumnya tetap dipertahankan:
- Dashboard dan follow up.
- Upload data besar.
- Marker berdasarkan Service Advisor.
- Cari dan pindahkan titik.
- Input alamat, link Google Maps, serta latitude dan longitude.
- Penyimpanan titik manual.

Pastikan kanan atas aplikasi menampilkan Versi 13.6 setelah deployment.
